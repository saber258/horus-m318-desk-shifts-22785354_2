# Horus M318 Desk Shifts Demo

Front desk shift scheduling demo for Horus M318 runtime.

## Features

- Shift scheduling with boundary validation
- Input validation for time ranges and overlapping shifts
- Test coverage for edge cases

## Getting Started

```bash
# Install dependencies
npm install

# Run tests
npm test
```

## Project Structure

- `src/` - Source code
- `tests/` - Test files
- `README.md` - This file
